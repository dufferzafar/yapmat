
# yapmat

* Commands:
    * yapmat manage [-n name] executable
    * yapmat list
    * yapmat logs [<id>|all]
    * yapmat status [<id>|all]
    * yapmat start|stop|restart <id> 
    * yapmat unmanage <id>

* Extra commands:
    * yapmat kill [signal-number] <id>
